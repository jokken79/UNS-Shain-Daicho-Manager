# Contribuyendo a UNS 社員台帳 Manager

¡Gracias por tu interés en contribuir!

## Cómo Reportar Bugs

- Usa GitHub Issues
- Describe el problema claramente
- Incluye versión de Python y OS
- Pasos para reproducir el error

## Cómo Sugerir Mejoras

- Abre un GitHub Issue
- Describe la mejora propuesta
- Explica por qué sería útil
- Proporciona ejemplos si es posible

## Pull Requests

1. Fork el repositorio
2. Crea una rama: `git checkout -b feature/nombre`
3. Haz commits con mensajes claros
4. Push a tu fork
5. Abre un Pull Request

## Guía de Estilo

- Usa PEP 8 para Python
- Añade docstrings a funciones
- Incluye type hints
- Mantén commits pequeños y descriptivos
